📋 Google Sheets Quraşdırma

Aşağıdakı addımları bir dəfə yerinə yetirin — bundan sonra hər qeydiyyat avtomatik olaraq Google Cədvəlinizə yazılacaq.

Addım 1 — Google Apps Script açın

script.google.com saytına daxil olun → "Yeni layihə" düyməsini basın.

Addım 2 — Kodu yapışdırın

Açılan redaktorda olan hər şeyi silin, aşağıdakı kodu yapışdırın:

function doPost(e) {
  var sheet = SpreadsheetApp
    .openById("BURAYA_SHEET_ID_YAZIN")
    .getActiveSheet();

  var data = JSON.parse(e.postData.contents);

  if (sheet.getLastRow() === 0) {
    sheet.appendRow([
      "№","Tarix","Saat","Ad","Soyad",
      "Telefon","Yaş","Kurs","Dərs Vaxtı","Qeyd"
    ]);
  }

  sheet.appendRow([
    sheet.getLastRow(),
    data.tarix, data.saat,
    data.ad, data.soyad,
    data.telefon, data.yas,
    data.kurs, data.vaxt,
    data.qeyd
  ]);

  return ContentService
    .createTextOutput(JSON.stringify({status:"ok"}))
    .setMimeType(ContentService.MimeType.JSON);
}

Addım 3 — Google Sheet ID-ni əlavə edin

Yeni bir Google Cədvəl yaradın. URL-dən Sheet ID-ni götürün:
https://docs.google.com/spreadsheets/d/BU_HİSSƏ_ID-dir/edit
Həmin ID-ni kodda BURAYA_SHEET_ID_YAZIN yerinə yazın.
